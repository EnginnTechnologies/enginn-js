Class: ResourceIndex

ResourceIndex(project)

Resource

A ResourceIndex is a collection of fetchable Resource.

It can be filtered and paginated through the chainable methods ResourceIndex#per, ResourceIndex#page, and ResourceIndex#where. It also provides methods to iterate on the collection.

Actual API requests are only issued when ResourceIndex#first or ResourceIndex#forEach methods are called. While ResourceIndex#forEach-ing, new API request will be issued when the end of a page is reached. Note that when using ResourceIndex#page, only the given page is reached for. One can also force fetching manually using ResourceIndex#fetch.
