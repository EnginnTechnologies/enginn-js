ResourcesIndex

A ResourcesIndex is a collection of fetchable Resource.

It can be filtered and paginated through the chainable methods ResourcesIndex#per, ResourcesIndex#page, and ResourcesIndex#where. It also provides methods to iterate on the collection.

Actual API requests are only issued when ResourcesIndex#first or ResourcesIndex#forEach methods are called. While ResourcesIndex#forEach-ing, new API request will be issued when the end of a page is reached. Note that when using ResourcesIndex#page, only the given page is reached for. One can also force fetching manually using ResourcesIndex#fetch.
